# SECTION B: ALGEBRA (22 marks)

**(a)** $\frac{(2x^3)^4}{4x^5}$

**Working:**
- Numerator: (2x³)⁴ = 2⁴ × x¹² = 16x¹² **[M1]**
- = 16x¹² ÷ 4x⁵ **[M1]**
- = 4x⁷ **[A1]**

**Answer:** 4x⁷

**(b)** $(27a^6b^{-3})^{\frac{2}{3}}$

**Working:**
- = 27^(2/3) × a^(6×2/3) × b^(-3×2/3) **[M1]**
- = 9 × a⁴ × b⁻² **[M1]**
- = $\frac{9a^4}{b^2}$ **[A1]**

**Answer:** $\frac{9a^4}{b^2}$ or 9a⁴b⁻²

## Question 4

**(a)** $\frac{x^2 - 5x + 6}{x^2 - 4}$

**Working:**
- Numerator: (x - 2)(x - 3) **[M1]**
- Denominator: (x + 2)(x - 2) **[M1]**
- Cancel (x - 2): = $\frac{x-3}{x+2}$ **[A1]**

**Answer:** $\frac{x-3}{x+2}$

---

**(b)** $\frac{3}{x^2 - 1} - \frac{2}{x + 1}$

**Working:**
- x² - 1 = (x+1)(x-1) **[M1]**
- = $\frac{3}{(x+1)(x-1)} - \frac{2(x-1)}{(x+1)(x-1)}$ **[M1]**
- = $\frac{3 - 2(x-1)}{(x+1)(x-1)}$ **[M1]**
- = $\frac{3 - 2x + 2}{(x+1)(x-1)} = \frac{5-2x}{x^2-1}$ **[A1]**

**Answer:** $\frac{5-2x}{x^2-1}$ or $\frac{5-2x}{(x+1)(x-1)}$

## Question 5

**(a)** $\sqrt{3x - 2} = x - 2$

**Working:**
- Square both sides: 3x - 2 = (x - 2)² **[M1]**
- 3x - 2 = x² - 4x + 4 **[M1]**
- x² - 7x + 6 = 0
- (x - 1)(x - 6) = 0 **[M1]**
- x = 1 or x = 6
- Check x = 1: √(3-2) = √1 = 1, but x - 2 = -1. Invalid.
- Check x = 6: √(18-2) = √16 = 4, and x - 2 = 4. Valid. **[A1]**

**Answer:** x = 6 only

# SECTION C: GRAPHS (16 marks)

## Question 7

**(a)** Line through A(-2, 7) and B(4, -5)

**Working:**
- Gradient = (-5 - 7)/(4 - (-2)) = -12/6 = -2 **[M1]**
- y - 7 = -2(x - (-2)) **[M1]**
- y - 7 = -2x - 4
- y = -2x + 3 **[M1]**
- 2x + y = 3 **[A1]**

**Answer:** 2x + y = 3

---

**(b)** Perpendicular line

**Working:**
- L₁: 3x - 2y = 8, rearrange: y = (3/2)x - 4, gradient = 3/2 **[M1]**
- Perpendicular gradient = -2/3 **[M1]**
- Through (6, 1): y - 1 = -2/3(x - 6) **[M1]**
- y - 1 = -2x/3 + 4
- y = -2x/3 + 5 **[A1]**

**Answer:** y = -⅔x + 5 (or y = -0.67x + 5)

## Question 8

**(a)** Intersection of y = 2x - 3 and y = x² - 4x + 3

**Working:**
- 2x - 3 = x² - 4x + 3 **[M1]**
- x² - 6x + 6 = 0 **[M1]**
- x = (6 ± √(36-24))/2 = (6 ± √12)/2 **[M1]**
- x = (6 ± 2√3)/2 = 3 ± √3
- x = 3 + √3 ≈ 4.73, y = 2(4.73) - 3 = 6.46 **[A1]**
- x = 3 - √3 ≈ 1.27, y = 2(1.27) - 3 = -0.46 **[A1]**

**Answer:** (3 + √3, 3 + 2√3) and (3 - √3, 3 - 2√3)
Or approximately (4.73, 6.46) and (1.27, -0.46)

---

**(b)** Solve x² - 4x + 3 < 2x - 3

**Working:**
- From (a), curve is below line between intersection points **[M1]**
- x² - 6x + 6 < 0 **[M1]**
- 3 - √3 < x < 3 + √3 **[A1]**

**Answer:** 3 - √3 < x < 3 + √3 (or 1.27 < x < 4.73)

# SECTION D: SHAPE AND SPACE (18 marks)

## Question 9

**(a)** Space diagonal of cuboid

**Working:**
- First find face diagonal: d₁² = 6² + 8² = 100, d₁ = 10 **[M1]**
- Space diagonal: D² = 10² + 10² = 200 **[M1]**
- D = √200 = 10√2 = 14.1 cm **[A1]**

Alternative: D = √(6² + 8² + 10²) = √200 = 14.1 cm

**Answer:** 14.1 cm (or 10√2 cm)

**(b)** Volume of cone

**Working:**
- Height: h² + 5² = 13² **[M1]**
- h² = 169 - 25 = 144
- h = 12 cm **[M1]**
- Volume = ⅓πr²h = ⅓ × π × 25 × 12 **[M1]**
- = 100π cm³ **[A1]**

**Answer:** 100π cm³

**(c)** Prove angle CAB = 90°

**Working:**
- AB = 15, AC = 20, BC = 25
- Check: 15² + 20² = 225 + 400 = 625 **[M1]**
- 25² = 625 **[M1]**
- Since AC² + AB² = BC², by the converse of Pythagoras' theorem, angle CAB = 90° **[A1]**

## Question 13

**(a)** $\frac{3}{x-2} + \frac{4}{x+1} = 2$

**Working:**
- Common denominator: (x-2)(x+1) **[M1]**
- 3(x+1) + 4(x-2) = 2(x-2)(x+1) **[M1]**
- 3x + 3 + 4x - 8 = 2(x² - x - 2) **[M1]**
- 7x - 5 = 2x² - 2x - 4
- 2x² - 9x + 1 = 0 **[M1]**
- x = (9 ± √(81-8))/4 = (9 ± √73)/4
- x = 4.39 or x = 0.11 **[A1]**

**Answer:** x = (9 + √73)/4 ≈ 4.39 or x = (9 - √73)/4 ≈ 0.11

- x² - 6x - 7 = 0 **[M1]**

**Answer:** x = 7 or x = -1
