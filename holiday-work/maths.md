## Y9 Consolidation Topics

These were covered in Y9 and may need reinforcement before Y10 builds on them.

### Quadratics
**Expanding brackets:** (x + 3)(x − 5) = x² − 2x − 15

**Factorising quadratics:** x² − 2x − 15 → (x − 5)(x + 3)
Steps: find two numbers that multiply to −15 and add to −2 → −5 and +3

**Solving quadratic equations:**
- By factorising: x² − 2x − 15 = 0 → (x − 5)(x + 3) = 0 → x = 5 or x = −3
- By quadratic formula: x = (−b ± √(b² − 4ac)) / 2a
- By completing the square: x² + 6x + 5 = 0 → (x + 3)² − 4 = 0 → x = −1 or x = −5

Practise all three methods. The quadratic formula always works — factorising is faster when it's clean.

Practice: Solve x² − 7x + 10 = 0 · 2x² + 5x − 3 = 0 · x² + 4x + 4 = 0 · 3x² − 12 = 0

### Simultaneous Equations
**Elimination method:** multiply one or both equations so coefficients match, then add or subtract.
**Substitution method:** rearrange one equation for one variable, substitute into the other.
**Graphical method:** plot both lines; intersection = solution.

Practise when to use which. Non-linear simultaneous equations (one linear, one quadratic) are Y10 content — preview below.

### Trigonometry — SOH CAH TOA

For right-angled triangles:
- **S**in θ = **O**pposite / **H**ypotenuse
- **C**os θ = **A**djacent / **H**ypotenuse
- **T**an θ = **O**pposite / **A**djacent

Process: identify which sides are involved → choose the correct ratio → rearrange to find the unknown.

Example: find angle θ if opposite = 5, hypotenuse = 10
sin θ = 5/10 = 0.5 → θ = sin⁻¹(0.5) = 30°

Practice: find missing sides and angles in 6 right-angled triangles.

### Circle Theorems (8 theorems to know)

1. Angle at the centre = 2 × angle at the circumference (same arc)
2. Angles in the same segment are equal
3. Angle in a semicircle = 90°
4. Opposite angles in a cyclic quadrilateral add to 180°
5. Tangent meets radius at 90°
6. Two tangents from an external point are equal in length
7. Alternate segment theorem: angle between tangent and chord = angle in alternate segment
8. Perpendicular from centre to chord bisects the chord

For every circle theorem question: state the theorem you are using by name. Marks are awarded for the reason, not just the answer.

Practice: 5 circle theorem questions from GCSE past papers. Write the theorem name for every step.

### Algebra — Rearranging Formulae

Rearrange to make a different variable the subject.

Example: v = u + at → make a the subject
v − u = at → a = (v − u) / t

Harder: v² = u² + 2as → make s the subject
v² − u² = 2as → s = (v² − u²) / 2a

Practice: rearrange 8 standard physics-style formulae — these appear in both Maths and Physics papers.

## Y10 GCSE Preview Topics

### Sine and Cosine Rules (non-right-angled triangles)

**Sine Rule:** a/sin A = b/sin B = c/sin C (use when you have a side and its opposite angle)
**Cosine Rule:** a² = b² + c² − 2bc cos A (use when you have all three sides or two sides and included angle)
**Area of a triangle:** Area = ½ ab sin C

---

### Vectors
A vector has **magnitude** (size) and **direction**. Written as a column vector or in bold/underline notation.

Vector addition: a + b (head-to-tail rule)
Scalar multiplication: 2a = twice as long, same direction

Path between two points: to get from A to B via known vectors — work out the route using addition and subtraction.

Parallel vectors: b is parallel to a if b = ka for some scalar k.

---

### Probability — Combined Events
- **Independent events:** P(A and B) = P(A) × P(B)
- **Mutually exclusive events:** P(A or B) = P(A) + P(B)
- **Tree diagrams:** multiply along branches (AND); add between branches (OR)
- **Conditional probability:** P(A given B has occurred) — adjust the denominator

**Venn diagrams:** use for problems involving sets and overlapping outcomes.

---

### Statistics — Higher Topics
- **Histograms:** frequency density = frequency ÷ class width (y-axis is not frequency)
- **Cumulative frequency graphs:** plot running total; read off median (50%), LQ (25%), UQ (75%); calculate IQR
- **Box plots:** minimum, LQ, median, UQ, maximum — useful for comparing distributions

---

### Non-Linear Simultaneous Equations (Y10 preview)
One equation is linear, one is quadratic. Substitute the linear equation into the quadratic.

Example: y = 2x + 1 and y = x² − 1
Substitute: 2x + 1 = x² − 1 → x² − 2x − 2 = 0 → solve by quadratic formula

---

### Upper and Lower Bounds
Any measurement given to a degree of accuracy has a range of possible true values.
- A length of 12.4 cm (to 1 d.p.) has bounds: 12.35 ≤ l < 12.45
- For addition/multiplication: UB of result = UB of each value combined
- For subtraction/division: UB of result = UB of first − LB of second (or ÷ LB)
